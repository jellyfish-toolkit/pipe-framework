class pipe.core.base.Pipe(request)[source]¶

Main structure in the framework. Represent pipe through which all data pass.

Pipe structure. Contains two parts - pipe for request and pipe for response. Data goes in next way request -> request extractor -> request transformer -> request loader -> response extractor -> response transformer -> response loader
